# Deteccion De Polaridad
Lo que se está haciendo en este proyecto es una detección de polaridad, una técnica que clasifica el texto según el sentimiento del usuario.

Resumiendo los puntos clave:

Clasificación: El código separa los tuits en tres grupos: positivo, negativo y neutral.

Tendencia: La gráfica de pastel revela que el 63% de los comentarios son negativos, lo que muestra una clara insatisfacción general.

Comparativa: Las gráficas de barras permiten identificar cuáles aerolíneas reciben más "hate" (como United) y cuáles tienen una mejor reputación (como Virgin America).

Confianza: El sistema mide qué tan seguro está de su análisis, siendo más preciso al detectar quejas que cumplidos.
